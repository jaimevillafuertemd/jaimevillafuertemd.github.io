Hay una variación a la eti­queta link.

Esta variación se pro­duce cuando quer­e­mos ver toda la ruta de una página web con un enlace a si misma.

Por ejem­plo si quiero poner el enlace al blog de COVSA:

Ten­dría que escribir así https://cor­po­ra­cioncovsa.blogspot.com/.

Notar que el enlace va entre llaves, no entre corchetes.
